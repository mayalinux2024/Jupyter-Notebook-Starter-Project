# Jupyter-Notebook-Starter-Project

## Overview

This project demonstrates basic data analysis using Jupyter Notebook, Pandas, and Matplotlib.

## Technologies

- Python 3.12
- Jupyter Notebook
- Pandas
- Matplotlib

## Features

- Creates a DataFrame
- Displays tabular data
- Generates a bar chart
